# su_ihtiyaci_hesaplama_uygulamasi


Bu proje, kullanıcının kilosu ve cinsiyet bilgisine göre günlük su ihtiyacını hesaplayan basit bir Python uygulamasıdır.


- Kullanıcıdan kilo bilgisi alır
- Cinsiyet bilgisine göre su ihtiyacını hesaplar
- Basit ve anlaşılır Python kodu
- Konsol üzerinden çalışır


Terminal veya komut satırında aşağıdaki komutu kullanabilirsiniz:

```bash
python su_ihtiyaci_hesaplama.py
